# LetDie
LetDie - плагин для Bukkit серверов, который делает так, что теперь зеленые твари криперы будут гореть днем.
Версия: v2.0.0
Особенности:
1.Относительная простота самого плагина.
2.Легкость в использовании.
Работоспособность:
Это проверялось на версиях:
1.6.4
